  • HTML is a computer language used on the Internet. This language is used to create web pages. The acronym stands for HyperText Markup Language, which in French means "hypertext markup language". This meaning is aptly named since indeed this language makes it possible to produce hypertext based on a markup structure.
    Start learning HTML now
  • The term CSS is the acronym for Cascading Style Sheets which can be translated as "cascading style sheets". CSS is a computer language used on the Internet to format HTML or XML files. Thus, style sheets, also called CSS files, include code that allows you to manage the design of a page in HTML.
    Start learning CSS now
  • JavaScript is a computer language used on web pages. This language has the particularity of being activated on the client workstation, in other words it is your computer which will receive the code and which will have to execute it. This is in opposition to other languages ​​which are enabled on the server side. The code is executed by your internet browser such as Firefox or Internet Explorer.
    Start learning JavaScript now
